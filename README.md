# mk-teamwork-server

#### 介绍
mk-teamwork是任务协同项目管理系统，

#### 软件架构
JAVA 1.8
MYSQL 8
基于前后端分离架构，服务端主要技术：springboot 、jwt  前端主要是vue;

#### 安装教程

1.  下载代码、编译打包，部署后端服务
2.  部署前端服务
3.  安装数据库

前端地址：https://gitee.com/wulon/mk-teamwork-ui
